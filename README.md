# cs3219-TaskA

### Instructions:
Clone the Github Repository on your local machine.

```
git clone https://github.com/candyhy/cs3219-TaskA.git
```

Navigate to the project directory.

```
cd cs3219-TaskA
```

Build docker images:

```
docker-compose build
```

Serve web app on localhost:

```
docker-compose up
```
